# erpbakery — Near Bakery & Co. ERP

Sistem Enterprise Resource Planning untuk toko roti Near Bakery & Co.

## Fitur

- 📊 Manajemen Bahan Baku & HPP
- 📝 Formulasi Resep Produk
- 💰 Simulasi Harga & Margin
- 🏭 Bill of Materials & Production Scheduling
- 📦 Inventori Multi-Gudang & FEFO
- 🚚 Logistik Distribusi Internal
- 🛒 Omnichannel POS & CRM
- 🔬 R&D Sandbox & Waste Control
- 📈 Laporan Keuangan & BI
- 🤖 AI Marketing Assistant

## Cara Install & Jalankan

```bash
npm install
npm run dev
```

Akses di http://localhost:3000

## Deploy ke Vercel

```bash
npm run build
```

Upload folder `dist/` ke Vercel.
